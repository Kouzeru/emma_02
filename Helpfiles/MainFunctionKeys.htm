Function and Hot Keys

The following function and hot keys can be used in any of the Emma 02 windows in some cases the function is depending on the emulated computer.

To change the defined keys below use menu item 'Settings / Function and Hot Keys'. In the Function and Hot Key Mapping window any of the PC keys can be changed and the 'Exit key' feature can be enabled. To change a key first select the button to be changed, after this 'Press Key' is shown on the button, now press the PC key it should be mapped to. Any PC key that is mapped twice will be shown in a red colour. Any key used by one of the emulators is shown in orange. Avoid double mapping (red) as the emulator will not be able to handle this. Re-using emulator keys (orange) will wotk but it will disable proper working of the same key in some of the emulators. Using SHIFT will in most cases still work in the emulators, i.e. when enabling ESC as exit key, SHIFT ESC will still work on a COMX as the COMX ESC key.
