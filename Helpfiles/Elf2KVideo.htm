Video Types

To select the specific Elf 2000 video type use the choice button 'Video'. Currently 2 types are supported: 1861 Pixie graphics and the intel 8275.

1861 Pixie graphics

1861 Pixie graphics is the same graphic system used for the Studio II computer giving a black and white screen of 64 by 128 large pixels.

Intel 8275

The intel 8275 is a video chip for text based output. Different screen sizes can be emulated via software setings of the intel 8275 registers. The Elf 2000 firmware supports this vide type but only when a PS2 keyboard is emulated at the same time.
