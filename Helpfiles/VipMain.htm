Cosmac VIP Emulator

This emulator will emulate the Cosmac VIP computer. Additional information about the Cosmac VIP can be found on the Emma 02 site.

After starting the Cosmac VIP emulator the 'F12' key can be used to simulate the Vip run switch, i.e. when 'F12' is pressed the VIP will stop execution and when pressed again it will restart. This can be used to start the VIP in system ROM mode: start the emulator, after this press the 'C' key and while keeping the 'C' pressed, press 'F12' twice. See also Function and Hot Keys.
