Colour and Sound Board

To use the VP590 Colour Board just check the VP590 checkbox, software included that uses the VP590 is the game blockout (Blockout [Steve Houk].c8x) and colour kaleidoscope (Color Kaleidoscope [Steve Houk, 1978].c8x) both running on CHIP-8X. Both files can be found in the directory 'Chip-8Chip-8X and Hybrids' (located in the application data directory, see Directory and File Structure)

For sound options use the Sound choice box to select:
- Beeper: For simple 'beep' function
- VP595 Simple Sound
- VP550 Supersound
- VP551 Supersound

Software included that uses the VP595 is the game blockout running on CHIP-8X.

Software included that uses the VP550 is the pin-8 software (pin-8.ram or yankee.ram). Optionally stereo sound can be switched on or off, when using stereo the channel 'A' sound is mapped to the left and channel 'B' to the right. Note that the VP550 implementation does NOT support the optional PAIA Drum Computer.

Software included that uses the VP551 is the 4 channel pin-8 software (pin-8.4ch.ram). Optionally stereo sound can be switched on or off, when using stereo the channel 'A' and 'C' sound is mapped to the left and channel 'B' and 'D' to the right. Note that the VP551 implementation does NOT support the optional PAIA Drum Computer.

The VP550 and/or VP551 music speed can be adapted by sliding the tempo control. This will either increase or decrease the speed at which the emulator sends the interrupt signals to the 1802 CPU.
