Auto Boot

To enable auto boot select the check box 'Auto Boot'. In auto boot mode the CDP18S600 will run automatically starting at the boot address 0 or 8000 depending on the selected boot type (program at 0, utility at 8000). To start the CDP18S600 in UT71 select boot type 'Utility'.

When auto boot is disabled the processor will not be started, in this case the processor needs to be started manual with the RUN U or RUN P buttons on the CDP18S640 front panel.
