Configurations

After initial start-up of Emma 02 default configurations will be made available according to the sub chapters below. These files can be re-installed at any time by using the menu item 'File/Re-install/Configuration Files'.

To load a pre defined computer configurations select one of items under the menu 'File/Configuration/Load'. Some of the configurations included in Emma 02 are described n the chapters below. For all computers the default setting is included in the 'Default' menu option.

To save a specific computer configuration, first make sure all setting for that computer are correct. Then select 'File/Configuration/Save'. A new window will open where it is possible to specify the 'Item Name' and optionally the 'Sub Menu Name'. After pressing save the new configuration will be saved under the specified menu. If a configuration was loaded before the save request, a new window will open to confirm overwrite, to replace the old menu or to create a new one.

The following settings are NOT saved in individual computer configurations but will be saved when saving the Emma 02 configuration on exit or manually when using menu item 'File/Configuration/Save All':

  • Any video setup items like colours and border sizes
  • Any key mapping definition
  • Superboard online settings like player name, location, email, home page, bookmarks and aliases

To delete a configuration from the menu select 'File/Configuration/Delete' and the configuration to be deleted.

Note: when installing a new version of Emma 02 the configuration menus will NOT be re-installed automatically. If so desired copy the latest files by using the menu item 'File/Re-install/COnfiguration Files'.

The configuration files location is:

  • Windows Vista and later: C:\Users\username\AppData\Roaming\Emma 02 Config
  • XP: C:\Documents and Settings\username\Application Data\Emma 02 Config
  • Mac OSX: ~/Library/Preferences/Emma 02
  • Linux: ~/.emma_02

Two EPROM board configurations are supported, one including a mini monitor and Quest Super BASIC REL 5.0 and one with the Super Gold Monitor and assembler. Both ROM files were created by Eric Tchong. Music Machine

Basic configuration to run the Music Maker and Christmas songs as all published in the Ipso Facto (Issue 5 and 8). Default the 'Music Maker.hex' and song 'O Christmas Tree.hex' will be loaded. Note that some songs will need a different speed which is indicated in the song file name (Double, 1_2 etc.), to play these also load the equivalent Music Maker file. The Ipso Facto articles can be found on www.cosmacelf.com and are also included in the Elf/Music directory.

CDP18S693 MCDS

RCA's low cost Microboard Computer Developmen System (MCDS) CDP18S693. This system includes the CDP18S601 CPU board and the CDP18S652 Memory & tape board. The CDP18S601 board includes 4K RAM but no ROM. The CDP18S652 includes the UT62 from 8000-87FF, BASIC 3 from B000-DFFF and a 1K RAM at 8C00.

CDP18S694 MCDS

RCA's Higher-Performance Microboard Computer Developmen System (MCDS) CDP18S694. This system includes the CDP18S601 CPU board and the CDP18S652 Memory & tape board. The CDP18S601 board includes 4K RAM but no ROM. The CDP18S652 includes the UT62 from 8000-87FF, Assembler / Editor from 900-AFFF, BASIC 3 from B000-DFFF and a 1K RAM at 8C00.

This configuration should be identical to the MCDS emulator however the implementation in Emma 02 is slightly different.
