ROM Files

To run the Cosmac Vip II 'ED' emulator at least the FPB and Cosmac Vip system ROM files should be specified in the 'ROM 1' (fpb.rom) and 'ROM 2' (vip.rom) fields.

To run the Cosmac Vip II 'RCA' emulator at least the vipii_2020 system ROM files should be specified in the 'ROM 1' field. ROM 2 is not available in 'RCA' mode.

Default Cosmac Vip II ROM location is the Vip directory (located in the application data directory, see Directory and File Structure). To change the location use the 'ROM 1' or 'ROM 2' button to browse for the file, do the same to change the filename or type in a new file name in the text field.

The binary, Intel hex and RCS Elf Emulator hex format are supported.
