Sound

Sound emulation volume can be changed by dragging the volume slider, sound can be turned off by dragging the slider to the far left. Equalization can be change via the main settings menu, default is 'TV Speaker'.

Some computer have support for different sound emulation, see following sub-chapter.

Beeper (hardware oscilator)

A very basic sound emulation for just one tone frequency (selectable in Hz). The tone will be heard when the Q flag is set by 1.

Supported for: Cosmac VIP, Cosmac VIP II, Telmac 1800, Cosmac Elf, Netronics Elf II and Quest Super Elf.

Note that the sound menu for the Cosmac VIP should be set to 'beeper' and for all Elf computers the Q Sound should be set to 'Hardware' and no Video Terminal should be configured to support this emulation.

Beeper (software oscilator)

A very basic sound emulation where the changing of the Q flag generates the tone oscilator. The speed of changing the Q flag in software determines the frequency.

Supported for: Oscom Nano, Cosmicos, Cosmac Elf 2000, Cosmac Elf, Netronics Elf II and Quest Super Elf.

Note that for all Elf computers the Q Sound should be set to 'Software' and no Video Terminal should be configured to support this emulation. For the Oscom Nano Sound should be set to 'Q' and for the Cosmicos NO CDP 1864 should be selected as video option.

Beeper (warpy)

A very basic sound emulation for just one tone with a decaying frequency. The tone will be heard when the Q flag is set by 1. This variant does not generate a fixed tone, instead the tone starts at 480 Hz and decays to 240 Hz quickly causing a warpy efect.

Supported for: RCA Studio II, Visicom COM-100.

CDP 1864 Sound

Supported for: Oscom Nano, Cosmac VIP, Cosmicos, RCA Studio III / Victory MPT-02, Telmac 2000 and ETI 660.

Note that the sound menu for the Cosmac VIP should be set to 'VP595 Simple Sound Board'. For the Oscom Nano Sound should be set to 'CDP 1864' and for the Cosmicos the CDP 1864 should be selected as video option.

CDP 1869 Sound

Supported for: COMX-35, TMC-600 and Pecom 64.
