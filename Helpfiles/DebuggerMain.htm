Debugger

The debugger gives you an inside view of what is going on in the Emma 02 emulator. To use the debugger select the 'Debugger' tab.

The debugger consists of 5 parts:

  1. 'Messages' containing information about the current running computer
  2. 'Direct Assembler' to assemble and disassemble 1802 or pseudo code directly from and to memory
  3. 'Debug 1802' to perform tracing commands on the 1802 CPU
  4. 'Debug Pseudo' to perform tracing commands on Chip-8, ST2, ST4, FEL and CARDTRAN
  5. 'Memory Dump' to inspect and change the CPU and video memory.

all on the running 1802 computer.

Note: None of the settings in the debugger windows are saved in the configuration, so at every start of the Emma 02 emulator all debugger settings are set to default.
