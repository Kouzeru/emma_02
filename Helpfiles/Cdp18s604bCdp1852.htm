CDP1852 Window

The CDP1852 Window can be used to simpulate the CDP1852 I/O chip. The window can be activate or deactivated via the 'CDP1852 Window' checkbox. A press on the INT button will send an interrupt signal to the emulated CPU and also activate EF2.

The CDP1852 is defined on I/O group 8 with:

  • Output 2: write to port, input 2: read port
  • EF 2: STB
