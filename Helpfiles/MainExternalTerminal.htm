External Terminal

To use Emma 02 with an external terminal additional software is needed. So far I have only been able to get this to work on Windows and Linux.

Of course serial terminal software is needed. For windows I have used and tested ZOC7. For Ubuntu I have tested with Minicom. Any other serial terminal emulation should work

Both Emma 02 and serial terminal software will communicate with a serial port on the host PC. So one option to use this would be to connect the two serial ports with a cable; I have not even tried this! Instead I recommend to use a serial port emulator. The serial port emulator will create two virtual ports on the host PC and connect them. One of the ports should be specified in Emma 02 (in Terminal Setup window) and the second in the serial terminal.

On windows I have tested both the free Null-modem emulator (com0com) and the Virtual Serial Port Driver from Eltima Software.

On Linux I have tested the free Null-modem emulator (tty0tty).

Terminal emulation is NOT possible when using intel 8275 emulation on the Elf 2000.

Following sub chapters describe terminal options, not all are available depending on emulated computer.

Serial mode

This mode is using serial I/O via the Q line and one of the EF lines. To use this mode make sure that the checkbox Uart 16450 is NOT selected (in Terminal Setup window).

Note that in this mode the transmit and receive (choice button T/R) baudrates need to be equal and, the speed options are depending on the selected computer.

Serial mode is not available for the MS 2000.

Uart

Two different uart are available: the 16450 for the Elf 2000 and MS2000 and the CDP1854 for the different Elf computers.

This uart mode can be use by selecting the Uart 16450 or Uart CDP1854 checkbox (in Terminal Setup window). Only basic usage of the 16450 and a simple mode 1 emulation for the CDP1854 is implemented.

In this mode the transmit and (T) and receive (R) baudrates can be different and the speed can be set between 50 to 19200 baud.

Reversed EF / Q

To use the terminal with reversed EF input and/or reversed Q output select the applicable checkbox (in Terminal Setup window).
