Diagnostic Board

To run the COMX emulator with the Diagnostic Board select checkbox 'Diagboard'. The choice box can be set to 'on' to start the diagnostic SW or to 'off' to start BASIC with the diagnostic board connected.

By default the following ROMs will be selected:

  • diag_pal1.bin, mapped to 0000-0FFF instead of BASIC ROM (if the diagnostic board is set to 'on'). BASIC ROM will be switched back on the first INP 1 instruction.
  • diag_pal2_v1.1.bin, mapped to C0000-D7FF. This v1.1 version will perform the extended ROM check on BASIC ROM file comx35.1.1.bin.
Next to this additional RAM will be selected from D800-DFFF.

The following alternative ROMs can be used:

  • diag_pal1.bin, diag_pal2.bin, with extended ROM check for the original COMX BASIC ROM (comx35.bin).
  • diag_ntsc1.bin, diag_ntsc2_v1.1.bin, NTSC version with ROM check for comx35.1.1.bin
  • diag_ntsc1.bin, diag_ntsc2.bin, NTSC versiom with ROM check for the original COMX BASIC ROM (comx35.bin).
The NTSC ROM set will be selected automatically when an NTSC COMX is selcted. To change from 1.1 to original version use the 'setup' button.

Aditionally the following options can be selected in the setup window:

  • ROM checksum: on or off
  • Factory unit: yes or no, when the factory unit is selected additional keyboard repeat and debounce tests are done as well as an IDEN test where IDEN is connected to bit 2 of INP 1.
  • Cassette cables: connected or disconnected

The following PC keys are mapped:

  • S: Step
  • R: Repeat
  • A: Abort
