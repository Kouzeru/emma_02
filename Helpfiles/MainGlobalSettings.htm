Global Settings & Menu

The following chapters describe all global settings and menu options; chapter headings are equal to the menu structure.

File

Configuration

Load

Load a specific configuration for the current selected computer. See chapter Configurations for more details.

Save

Save a specific configuration for the current selected computer. See chapter Configurations for more details.

Delete

Delete a specific configuration for the current selected computer. See chapter Configurations for more details.

Save All

Save all global and computer emulator specific settings.

Save All on Exit

If selected all global and computer emulator specific settings will be saved on exit of Emma 02.

Set Default (All)

Reset all computer emulator specific settings, 'Wav file and Real Cassette Settings' and 'Function and Hot Keys' back to Emma 02 defaults. Note remaining global settings, SB Player name, location, email address and aliasses will not be reset.

Re-install

Configuration Files

This will re-install the default configuration files for the computer configurations that can be loaded from the above described 'Configuration' menu. This might be needed to install updated configuration files from the latest Emma 02 release or if current installed files are corrupt.

When installing a new version of Emma 02 the configuration menus will NOT be re-installed automatically to avoid overwriting changes made by users.

1802 SW Files

This will re-install the default 1802 SW files as stored in the Emma 02 data directory. This might be needed to install updated 1802 SW files from the latest Emma 02 release or if current installed files are corrupt.

When installing a new version of Emma 02 the data directory will NOT be re-installed automatically to avoid overwriting changes made by users.

Change Data Directory

This will allow you to change the location of the data directory. The location for the data directory is default set to 'C:UsersusernameAppDataRoamingEmma 02' for Vista and 7, 'C:Documents and SettingsusernameApplication DataEmma 02' for XP and to '/Users/username/Documents/emma_02' for Mac OSX.

A small window will open where with the 'browse' button a new location can be selected. After this 2 options are available: either to only save the new location ('Save' button) or save and move the existing data directory and files to the new location ('Move & Save' button). If the save only option is used the data folder should be moved manually. When moving the folder to another disk the process could take a few minutes (depending on the disk speed). In all cases all settings will be set to default as in the 'Set Default Configuration' menu action.

Note: after a re-installation of Emma 02 (or installation of a later version) the data directory will be installed to the original default location.

See also chapter Directory and File Structure.

Settings

Wav file and Real Cassette Settings

Open window to change settings for wav files and real cassette handling, see Settings chapter in Real Cassette Support for details.

Equalization

Select one of the sub menu items to adjust sound equalization. Default 'TV Speaker' is selected. For additional details see the Sound chapter.

Window

Sub menu items for Emma 02 window handling, see chapter Window Settings.

CPU Type

Select one of the sub menu items to choose the CPU type used for all emulated computers. Default CDP1805 is selected allowing a more extensive instruction set than the CDP1802 which was used in most emulated computers.

Startup Values

Select one of the sub menu items to set the startup values of CPU registers, RAM and/or Video RAM. The following options can be chosen:

  • Zero: CPU registers, RAM content or Video RAM content is set to 0 on startup
  • Random: CPU registers, RAM content or Video RAM content is set to a random value on startup
  • Dynamic: RAM content or Video RAM content is set similar to the COMX Dynamic RAM content on startup. This content is a repeating patern of 64 values 255 (FF) followed by 64 values of 0, one random location of these 128 values will have a random value instead of 0 or 255.

Function and Hot Keys

Open function and Hot keys window, see also chapter Function and Hot Keys.

Use Numeric Keypad

Unselect this menu item if your PC does NOT have a numeric keypad (numpad). If unselected all keymap windows will not use any key definitions from the PC numeric keypad. Note that the keypad definition 'on Character' for keypad B in the VIP emulator will always use the numeric keypad as there is no good way to handle this without using a numeric keypad.

Default keyboard

Default keyboard mapping is defined for a US keyboard layout. However a few different other layouts (Dutch, German, French, Swedish / Finnish) are provided and can be selected via this menu. See chapter Key Mapping for more details.

Help

Help

COMX Superboard

Two sub menu items with direct links opening the COMX Superboard homepage and high score page.

Check for Update on Start-up

At start-up Emma 02 will check if a newer version is available. This will only work if an internet connection is available.

To disable the start-up check, uncheck this menu item.

Do a manual update check.
