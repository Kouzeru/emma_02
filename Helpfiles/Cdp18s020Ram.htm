RAM Size and Memory Protect

To specify the onboard RAM size of the emulated CDP18S020 Evaluation Kit (between 256 bytes and 4 Kb) use the 'RAM' spin button.

4 dip switches are available on the CDP18S020 front panel, switch these up to enable RAM memory protection (disable writing to RAM). Each dip switch controls 1 KB of RAM at address:

  • 000-3FF
  • 400-7FF
  • 800-BFF
  • C00-FFF
