RCA Studio III / Victory MPT-02 Emulator

This emulator will emulate the RCA Studio III / Victory MPT-02 computer. The Victory is a clone of the RCA Studio III which was never release by RCA. Additional information about the Victory can be found on the Emma 02 site.

This chapter includes:

Key Definition
ROM Files
Video Mode
MPT-02 Cartridges
Chip 8 Interpreter
40th Anniversary Multi-cart
