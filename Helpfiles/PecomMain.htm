Pecom 32 & 64 Emulator

This emulator will emulate the Pecom 32 or 64 home computer. Additional information about the Pecom 32 and 64 can be found on the Emma 02 site.

To run the Pecom comupter without DRAM refresh (like the open-source clone COMIX-35) make sure 'DRAM refresh' is not selected.

This chapter includes:

ROM Files
Keyboard Mapping
Printer Support
Cassette Support
