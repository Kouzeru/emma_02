ROM Files

To run the TMC-600 emulator at least the TMC-600 system and character ROM files should be specified in the 'ROM' and 'Character' fields. Default TMC-600 ROM location is the TMC600 directory (located in the application data directory, see Directory and File Structure). To change the location use the 'ROM' or 'Character' button to browse for the file, do the same to change the filename or type in a new file name in the text field.

Similar as for the system ROM the Expansion ROM can be specified via 'EXP ROM'.

The binary, Intel hex and RCS Elf Emulator hex format are supported. Also MESS multiple ROM files are supported; to use this just select one of the ROM files.
