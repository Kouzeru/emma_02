Auto Boot

To enable auto boot select the check box 'Auto Boot'. In auto boot mode the CDP18S020 Evaluation Kit will run automatically starting at the boot address 0 or hex 8000 depending on the selected boot type (program at 0, utility at hex 8000). To start the CDP18S020 in UT4 select boot type 'Utility'.

When auto boot is disabled the processor will not be started, in this case the processor needs to be started manual with the RUN U or RUN P buttons on the front panel.
