Rom and Ram Files

To run the Elf emulator you will need to load SW either as ROM or RAM. To load initial SW there are 2 ROM/RAM buttons, to specify which file to load use the 'ROM x' or 'RAM x' button to browse for the file, or type in a new file name in the text field. Default the Elf roms should be located in the Elf directory (located in the application data directory, see Directory and File Structure).

To load SW into RAM instead of ROM select the toggle ROM/RAM button (2 arrows) to switch a button from 'ROM x' to 'RAM x' and vice versa. When loading to RAM make sure RAM is defined for the required memory locations see also RAM Address.

Every of the 2 ROM/RAM files can be placed in a different directory if required; just browse for the file and the emulator will remember the location. Note that files in binary format, normally using extension .bin, will be loaded at address 0. This means loading 2 binary files will not be possible as they will all load at the same starting address. Intel hex and RCS Elf emulator hex format include storage locations and as such will be loaded in the correct address location. Make sure that if you use overlapping hex files that the file which you need to be complete in memory is loaded with the highest ROM/RAM button 1.
