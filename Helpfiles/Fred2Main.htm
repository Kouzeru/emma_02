FRED 2 Emulator

This emulator will emulate the FRED 2 computer. The FRED (Flexible Recreational and Educational Device) concept was used by RCA for multiple prototype computers eventually resulting in the release of computers like the VIP and Studio II. The FRED 2 emulated by Emma 02 is based on the information found in this manual as well as Paul Robson's FRED emulator. It is very similar to the FRED 1 but is running on the CDP 1801 instead of the SYSTEM 00 chip set and as such is able to run FEL-1. Additional information about the FRED 2 can be found on the Emma 02 site.
