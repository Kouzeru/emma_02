Front Panel

STEP - Single step the processor, will execute next CPU cycle and then enter Wait mode

RESET - Places the CPU into the reset state

RUN P - Execute program in RAM at address 0

RUN U - Execute program in ROM at address 8000

4 Dip switches - switch up to enable RAM memory protection (disable writing to RAM). Each dip switch controls 1 KB of RAM at address:

  1. 000-3FF
  2. 400-7FF
  3. 800-BFF
  4. C00-FFF

DATA BUS - 8 leds (7 to 0) showing the value of the data bus

MEMPRY ADDRESS - 16 leds (15 to 0) showing the value of the address bus

STATUS - CPU Status leds SC1/SC0 showing the state type, W for Wait, C for Clear and Q.
