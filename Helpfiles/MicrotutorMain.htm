Microtutor Emulator

This emulator will emulate the RCA Microtutor. A manual for the Microtutor can be found here: Microtutor Manual - MPM 109.

The Microtutor is similar to the Elf however simpler and using an RCA CDP1801 CPU. This is the only computer in Emma 02 that does not use the RCA CDP1802. The Microtutor emulation forces Emma 02 to run on the RCA CDP1801 even if the overall menu setting (Settings/CPU Type) is set to CDP1802, CDP1804 or CDP1805.
