Control Windows

The FRED control panel can be emulated by selecting the checkbox 'FRED Window'

Emulation of the leds can take a lot of PC CPU power so default these are only updated every 100 ms. This can be changed in the Emma 02 window by changing the 'Leds' field. When 0 ms (or nothing) is specfied the leds will be updated in real time.

Front Panel

The front panel includes the following leds and switches:

  • Bit leds: 8 bit leds (0-7) showing the 8 bit data bus on IDLE Rx instructions and run switch press (see below).
  • Stop led: Indicates that the FRED CPU is not running.
  • Ready led: Indicates that the FRED is in IDLE state.
  • Error led: On when a cassette parity error occurs.
  • Off switch: Turns the FRED power off.
  • Reset switch: If pressed resets the FRED. It will deactivate all I/O devices, set P=0 and R=0000. After reset stop and ready leds should both be on. Error led should be off. Note that when selecting 'Auto boot' a reset will auto boot the CPU directly.
  • Read switch: Activates the cassette read direct mode.
  • Run switch: If pressed will start the FRED in RUN mode starting instruction at R0. If the read or card switch is up the run switch will show the current value of address in R0 on the leds and increase R0 with one.
  • Card switch: not supported in the Emma 02 FRED emulator.
