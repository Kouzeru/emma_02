Front Panel

L - Places the computer into Load mode

R - Places the CPU into the reset state

G - Go, this starts the processor running

W - Enter Wait mode. This also removes memory protection in Load mode

M - Enable 32 byte mini monitor, this is in address 00-1f
Note: If the superboard (stored at hex 8000) or giantboard (stored at hex F000) monitor is loaded the 'M' button will execute that monitor instead of enabling the mini monitor.

S - Single step the processor, will execute next instruction and then enter Wait mode

P - Enables memory Protection in Load mode

I - Input, this is connected to DMA_IN in Load mode, or EF4 in run mode
